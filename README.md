# Wetube

- **global router**
  / -> home
  /join -> Join
  /login -> Login
  /search -> Search

- **user router**
  /users/:id -> See User
  /users/logout -> Log Out
  /users/edit -> Edit MY Profile
  /users/remove -> Remove MY Profile

- **video router**
  /videos/:id -> See Video
  /videos/:id/edit -> Edit Video
  /videos/:id/deleteVideo -> Delete Video
  /videos/upload -> Upload Video
